# 项目简介

## 核心功能
- 基于YOLOv8的布料缺陷检测系统
- 支持6类布料缺陷检测：带沙、断沙、面球、破洞、脱沙、污渍
- 包含2000张标注布料缺陷数据集
- 提供完整源码、训练好的模型及训练结果
- 支持图片、视频及摄像头实时检测
- 界面实时显示目标位置、目标总数和置信度
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 可选择检测模型

## 适用场景
- 布料生产线质量检测
- 纺织品制造过程中的缺陷识别
- 布料质量控制系统
- 工业自动化检测场景

## 技术特点
- 使用Python 3.9开发环境
- 集成opencv-python、PyQt5、ultralytics、torch1.9等库
- 采用PyCharm+Anaconda或VSCode+Anaconda开发环境
- 包含完整UI界面源文件和图标
- 提供项目运行教程和数据集
